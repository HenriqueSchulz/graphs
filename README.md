# Projeto de Grafos

## 📦 Bibliotecas Requisitadas

Instale as dependências necessárias com o comando:

```bash
pip install pyvis psutil
```

## ▶️ Execução

Para executar o programa, basta rodar o arquivo principal:

```bash
python main.py
```

## 🧩 Estrutura do Projeto

* **Node.py**
  Implementa um nó do grafo.

* **Graph.py**
  Implementa as funções do grafo.

* **Benchmark.py**
  Executa os testes necessários.

## 📊 Resultados

* Todos os resultados gráficos são gerados automaticamente na pasta:

```
results/
```

* As tabelas com os dados dos testes são exibidas diretamente no terminal durante a execução.
